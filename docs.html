Rideshare API Documentation

1. Kullanıcı Kayıt ve Giriş

Kullanıcı Kaydı

POST /api/auth/register
Headers: Content-Type: application/json
Body:
{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "password123"
}

Kullanıcı Girişi

POST /api/auth/login
Headers: Content-Type: application/json
Body:
{
  "email": "john@example.com",
  "password": "password123"
}

2. Kullanıcı Profili

Profil Bilgilerini Alma

GET /api/user
Headers: 
  x-auth-token: [token]

Profil Bilgilerini Güncelleme

PUT /api/user
Headers: 
  x-auth-token: [token]
Body:
{
  "name": "John Updated",
  "email": "john_updated@example.com"
}

3. Seyahat Oluşturma ve Yönetimi

Seyahat Oluşturma

POST /api/trips
Headers: 
  x-auth-token: [token]
Body:
{
  "origin": "Istanbul",
  "destination": "Ankara",
  "date": "2024-06-15T10:00:00Z",
  "availableSeats": 3,
  "price": 100
}

Tüm Seyahatleri Listeleme

GET /api/trips

Belirli Bir Seyahati Görüntüleme

GET /api/trips/:id

Seyahati Güncelleme

PUT /api/trips/:id
Headers: 
  x-auth-token: [token]
Body:
{
  "origin": "Istanbul",
  "destination": "Izmir",
  "date": "2024-06-16T10:00:00Z",
  "availableSeats": 2,
  "price": 120
}

Seyahati Silme

DELETE /api/trips/:id
Headers: 
  x-auth-token: [token]

4. Rezervasyon Sistemi

Rezervasyon Oluşturma

POST /api/bookings
Headers: 
  x-auth-token: [token]
Body:
{
  "tripId": "trip_id",
  "seats": 2
}

Tüm Rezervasyonları Listeleme

GET /api/bookings

Belirli Bir Rezervasyonu Görüntüleme

GET /api/bookings/:id

Rezervasyonu İptal Etme

DELETE /api/bookings/:id
Headers: 
  x-auth-token: [token]

5. Mesajlaşma Sistemi

Mesaj Gönderme

POST /api/messages
Headers: 
  x-auth-token: [token]
Body:
{
  "receiverId": "user_id",
  "content": "Hello, how are you?"
}

Kullanıcıya Ait Tüm Mesajları Listeleme

GET /api/messages
Headers: 
  x-auth-token: [token]

İki Kullanıcı Arasındaki Mesajları Listeleme

GET /api/messages/user/:userId
Headers: 
  x-auth-token: [token]
